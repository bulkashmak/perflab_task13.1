<h3>"Корыстный интерес преподавателя"</h3>

<p>Разработать утилиту, работающую с двумя файлами (используйте потоки ввода):</p>
<ol>
<li>текстовый файл вопросов (1 вопрос на строку, количество строк = количество вопросов)</li>
<li>текстовый файл имен учеников (1 ФИО на строку, количество строк = количество учеников)</li>
</ol>
<p>Утилита должна считать данные из этих файлов, сгенерировать пары ученик-вопрос случайным образом и записать в файл с именем <b>вопросы-%сегодняшняя_дата%.txt</b></p>
